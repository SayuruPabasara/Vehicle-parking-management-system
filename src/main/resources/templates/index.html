About ParkNow

ParkNow is a Vehicle Parking Management System developed as a Year 1 Semester 2 Java OOP project at Sri Lanka Institute of Information Technology. The project was created by a dedicated team of six members with the aim of improving traditional parking management through a more efficient digital solution.

This system allows users to easily manage parking activities by providing features such as parking slot reservations, vehicle entry tracking, availability monitoring, and administrative management functions. It helps reduce manual errors, save time, and improve the overall parking experience for both drivers and administrators.

Built using object-oriented programming principles, the project demonstrates practical implementation of Java concepts while solving a real-world problem through smart parking management.
